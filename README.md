# react-scroll-sync

Synced scroll position across multiple scrollable elements.

## Installation

```shell
$ npm install react-scroll-sync
```

## Usage

```html
<ScrollSync>
  <div style={{ display: 'flex', position: 'relative', height: 300 }}>
    <ScrollSyncPane>
      <div />
    </ScrollSyncPane>

    <ScrollSyncPane>
      <div />
    </ScrollSyncPane>
      
    <ScrollSyncPane>
      <div />
    </ScrollSyncPane>
  </div>
</ScrollSync>
```
